# 🎓 Student Management System (Python + OOP + Streamlit)

## 📌 Project Overview

This is a simple **Student Management System** built using **Python (OOP concepts)** and **Streamlit**.
It helps to manage student data like adding, viewing, updating, and deleting records.

---

## 🚀 Features

* Add new student
* View all students
* Update student details
* Delete student
* Simple and interactive UI using Streamlit

---

## 🛠️ Technologies Used

* Python
* Object-Oriented Programming (OOP)
* Streamlit

---

## 📂 Project Structure

```
project/
│── app.py              # Streamlit main file
│── student.py          # Student class (OOP)
│── data.json / db      # Data storage
│── README.md
│──student_manager.py
│──test_student.py
```

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/Parth1324355675/Student-management_System_with_Frontend_Backend
```

2. Go to project folder:

```
cd project
```

3. Install dependencies:

```
pip install streamlit
```

4. Run the app:

```
streamlit run app.py
```
